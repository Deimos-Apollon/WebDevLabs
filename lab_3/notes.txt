Объект юзера содержит следующие элементы:
- name - имя
- birth - дата рождения в формате дд/мм/гггг
- role - роль (admin - администратор, user - пользователь, guest - гость)
- status - статус (unconfirmed - не подтвержден, confirmed - подтвержден, banned - забанен)
- email - электронная почта
- id - идентификатор пользователя
- friends - массив id друзей пользователя
- img - ссылка на изображение

Объект сообщения содержит следующие элементы:
- author - имя автора
- img - аватар автора
- date - дата создания поста
- text - текст поста
- id - идентификатор поста
- user_id - идентификатор пользователя, написавшего сообщение
